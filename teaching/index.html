Teaching

Descriptions and materials for the courses I am giving. Description of the students' projects that I am supervising.

I am a teacher at HSE University and a member of a teaching team at LauzHack (EPFL). Besides, I am the coursework/thesis supervisor for several students. Below you can find the description of the courses and my students’ projects:

HSE

Deep Learning for Audio

The course covers various applications of Deep Learning for Audio and MLOps basics.

LauzHack (EPFL)

Deep Learning Bootcamp

Bootcamp covering introductory and advanced topics of Deep Learning. Twice per year (Summer and Winter).

Workshops

Workshops covering different Computer Science topics. During Fall and Spring semesters.

Supervision

Unified Voice Biometric System Against Spoofing Attacks

The project aims on the development of SASV system for all three types of attacks (imposter attack, synthesized speech, recorded speech).

Modern Voice Authentication Systems

The project aims on finding new ways to improve generalization of countermeasure systems.

Reproducibility of Synthetic Speech Detection Systems

The project aims on supporting research community by reproducing ADD papers without open-source code.

Synthesized Speech Detection Under Emotional Voice Conditions

The project aims on the development of a countermeasure system that can handle emotional speech input.

Speaker Verification System for Emotional Voice

The project aims on the development of a speaker verification system that can handle emotional speech input.

Signature Forgery Detection Using Deep Learning

The project aims on the development of a signature forgery detection system.

Alignment of Music Recordings With Their Scores

The project aims on the development of a forced alignment algorithm for music recordings and their scores.

Multimodal Systems for Deepfake Video Detection

The project aims on the development of an audio-visual deepfake detection system.

The Impact of Language on Automatic Speaker Verification Systems

The project explores how language changes the performance of a speaker verification system and aims to develop a system for non-English language.

Tabular Data Synthesis Using Generative Models

The project aims on supporting research community by reproducing papers without open-source code.

Creation of Audio Effects Using Deep Learning

The project explores the creation of audio effects using deep learning.
